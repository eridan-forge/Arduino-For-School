# Урок 1: Мигающий светодиод

Самый первый проект для начала.

## Цели урока:
1. Понять структуру программы Arduino
2. Научиться подключать светодиод
3. Освоить базовые команды

## Оборудование:
- Компьютер с Wokwi
- Виртуальная Arduino
- Виртуальные компоненты

## Ход урока:

### 1. Теория (10 минут)
- Что такое Arduino?
- Понятия: пин, код, светодиод, резистор

### 2. Практика (25 минут)
- Сборка схемы в Wokwi
- Написание кода
- Запуск симуляции

### 3. Эксперименты (10 минут)
- Изменение скорости мигания
- Добавление второго светодиода

# Методические заметки для учителя

## Типичные ошибки учеников:
1. ❌ Забывают pinMode() в setup()
2. ❌ Путают HIGH и LOW
3. ❌ Не понимают зачем delay() и как важно ";" для работы кода

## Как объяснять сложные понятия:
- Аналогия с краном и водой для тока
- Сравнение delay() с паузой в музыке

## Что нужно:
- Arduino Uno
- Светодиод
- Резистор 220 Ом

## Схема:
Подключи светодиод к пину 13 через резистор.

## Код:
Смотри файл blink.ino в этой папке.

## Ошибки:
Если не работает — проверь, что длинная нога светодиода подключена к резистору.
