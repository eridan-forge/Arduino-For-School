# Урок 1: Мигающий светодиод

Самый простой проект для начала.

## Что нужно:
- Arduino Uno
- Светодиод
- Резистор 220 Ом

## Схема:
Подключи светодиод к пину 13 через резистор.

## Код:
Смотри файл blink.ino в этой папке.

## Ошибки:
Если не работает — проверь, что длинная нога светодиода подключена к резистору.
[посмотреть готовый результат в симуляторе wowki](https://wokwi.com/projects/453758956373015553)
