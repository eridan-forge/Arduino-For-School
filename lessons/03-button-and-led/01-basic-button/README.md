# Простейшая кнопка

## Описание
Базовый пример работы с кнопкой без защиты от дребезга.
Показывает проблему множественных срабатываний.

## Схема подключения
Кнопка: D2 → кнопка → GND
Светодиод: D13 → LED(+) → резистор 220Ω → GND

## Проблема
При нажатии кнопки светодиод может переключаться несколько раз
из-за "дребезга контактов".
