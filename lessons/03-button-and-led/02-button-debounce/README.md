# Кнопка с защитой от дребезга

## Описание
Решение проблемы дребезга контактов с помощью флага и задержки.

## Что такое дребезг?
Когда вы нажимаете механическую кнопку, контакты соприкасаются
не сразу, а несколько раз за миллисекунды. Arduino успевает
считать каждое касание как отдельное нажатие.

## Решение
1. Используем флаг `buttonPressed` для отслеживания состояния
2. Добавляем задержку 300 мс после нажатия
3. Сбрасываем флаг при отпускании кнопки
