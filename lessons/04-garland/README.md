0–5 мин | ВСТУПЛЕНИЕ (честный разговор)
"Сегодня мы соберём устройство, которое похоже на гирлянду, но работает умнее. В настоящей ёлочной гирлянде лампочки соединены цепочкой: одна перегорела — всё погасло. Это называется последовательное соединение.

А наша гирлянда будет параллельной. Каждый светодиод живёт своей жизнью. Если один перестанет работать — остальные продолжат мигать. Именно так сделаны уличные гирлянды, светодиодные ленты и иллюминация на домах.

Почему это круто? Потому что мы можем управлять каждым светодиодом отдельно. У нас будет 5 лампочек и 4 режима работы. Переключать режимы будем одной кнопкой."

10–35 мин | КОД (разбор по режимам)
Режим 0: ЗВЁЗДНОЕ НЕБО (случайное мигание)
cpp
void mode0() {
  for (int i = 0; i < 5; i++) {
    if (millis() - prevMillis[i] >= interval[i]) {
      prevMillis[i] = millis();
      digitalWrite(leds[i], !digitalRead(leds[i]));
      interval[i] = random(200, 2000);
    }
  }
}
"Это как звёзды на небе: каждая мигает когда хочет. Мы уже делали такое."

Режим 1: ВСЕ ГОРЯТ
cpp
void mode1() {
  for (int i = 0; i < 5; i++) {
    digitalWrite(leds[i], HIGH);
  }
}
"Просто всё включено. Как обычная лампочка."

Режим 2: БЫСТРОЕ МИГАНИЕ
cpp
void mode2() {
  static unsigned long lastBlink = 0;
  static bool state = HIGH;
  
  if (millis() - lastBlink > 200) {
    lastBlink = millis();
    state = !state;
    for (int i = 0; i < 5; i++) {
      digitalWrite(leds[i], state);
    }
  }
}
"Все мигают вместе, как аварийка на машине."

Режим 3: БЕГУЩИЙ ОГОНЁК
cpp
void mode3() {
  static int currentLed = 0;
  static unsigned long lastMove = 0;
  
  if (millis() - lastMove > 100) {
    lastMove = millis();
    for (int i = 0; i < 5; i++) {
      digitalWrite(leds[i], LOW);
    }
    digitalWrite(leds[currentLed], HIGH);
    currentLed++;
    if (currentLed >= 5) currentLed = 0;
  }
}
"Это классика: огонёк бежит по кругу. Такие гирлянды продают в магазинах."

🧠 ПОЧЕМУ ЭТОТ УРОК СРАБОТАЕТ
Честность — ты объясняешь, почему это "умная гирлянда", а не обычная

Понятность — режимы простые и знакомые

Красота — бегущий огонёк дети обожают

Развитие — можно бесконечно усложнять (новые режимы, больше светодиодов)
