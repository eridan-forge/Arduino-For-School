## 🧠 Теория: как работает LCD экран

LCD (Liquid Crystal Display) — это **окно в твой проект**.  
В отличие от светодиода, он может показывать **текст, цифры и даже символы**.

Мы используем **LCD 1602** — 16 символов в строке, 2 строки.  
С **I2C модулем** на обратной стороне — нужно всего **4 провода** вместо 16.

### Что такое I2C?
Это шина для обмена данными. У каждого I2C-устройства есть свой **адрес**, например `0x27`.  
Ардуино обращается к устройству по адресу, как к абоненту в телефонной книге.

---

## 🔌 Подключение (4 провода)

| LCD I2C | Arduino Nano/Uno |
|---------|------------------|
| VCC     | 5V               |
| GND     | GND              |
| SDA     | A4               |
| SCL     | A5               |

> ⚠️ Если экран не загорается — покрути **синюю крутилку** на обратной стороне (контрастность).  
> Появятся квадратики — значит питание есть.

## 📚 Библиотека

### Как установить
1. Скетч → Подключить библиотеку → **Управлять библиотеками**
2. Найти: **LiquidCrystal I2C by Frank de Brabander** (или Marco Schwartz)
3. Нажать **Установить**

### Подключение в коде
```cpp
#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);  // адрес 0x27, 16 символов, 2 строки
🔧 Где взять адрес экрана
Если 0x27 не работает — возможно адрес 0x3F.

Узнать адрес можно скетчем:

Скетч → Примеры → Wire → i2c_scanner
Загрузи → открой Монитор порта → увидишь Found address: 0x27

💻 Основные команды (шпаргалка)
cpp
lcd.init();               // включить экран
lcd.backlight();          // включить подсветку
lcd.clear();              // очистить экран
lcd.setCursor(колонка, строка);  // 0–15, 0–1
lcd.print("Текст");       // напечатать
lcd.print(123);           // можно и числа

Основной код
cpp
#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);  // если не работает — попробуй 0x3F

void setup() {
  lcd.init();           // включить экран
  lcd.backlight();      // зажечь подсветку
  lcd.setCursor(0, 0);  // в начало первой строки
  lcd.print("Hello!");  // печатаем
  lcd.setCursor(0, 1);  // в начало второй строки
  lcd.print("LCD works!");
}

void loop() {
  // здесь пока пусто
}
