## 🧠 Теория: как летучая мышь видит мир

HC‑SR04 работает по принципу **эхолокации** — как летучие мыши или дельфины.

- Датчик **посылает** ультразвуковой сигнал (мы его не слышим).
- Сигнал отражается от препятствия и **возвращается**.
- Чем дольше сигнал шёл туда и обратно — тем дальше объект.

Скорость звука в воздухе ≈ **340 м/с** или **0.034 см/мкс** (микросекунду).

Формула:
расстояние = (время звука × скорость звука) / 2

text

Делим на 2, потому что звук летит туда **и обратно**.

## 🔌 Подключение (4 провода)

| HC‑SR04 | Arduino Nano/Uno |
|---------|------------------|
| VCC     | 5V               |
| GND     | GND              |
| Trig    | D9               |
| Echo    | D10              |

> ⚠️ **Важно:** не перепутай Trig и Echo. Если датчик не работает — проверь питание и контакты.

## 💻 Основные команды

digitalWrite(Trig, LOW);
delayMicroseconds(2);         // пауза для стабильности
digitalWrite(Trig, HIGH);
delayMicroseconds(10);        // импульс 10 мкс
digitalWrite(Trig, LOW);

long duration = pulseIn(Echo, HIGH);   // ждём сигнал, считаем время
float distance = duration * 0.034 / 2; // переводим в сантиметры


Что такое pulseIn?
pulseIn(пин, HIGH) ждёт, когда на пине появится высокий сигнал, и считает, сколько микросекунд он держится.
Это и есть время полёта звука.

15–30 мин | Пишем код
Полный код


#define TRIG 9
#define ECHO 10

void setup() {
  Serial.begin(9600);
  pinMode(TRIG, OUTPUT);
  pinMode(ECHO, INPUT);
  Serial.println("Датчик запущен");
}

void loop() {
  // отправить короткий импульс
  digitalWrite(TRIG, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG, LOW);

  // измерить время ответа
  long duration = pulseIn(ECHO, HIGH);

  // перевести в сантиметры
  float distance = duration * 0.034 / 2;

  // вывести в монитор порта
  Serial.print("Расстояние: ");
  Serial.print(distance);
  Serial.println(" см");

  delay(500);
}



Загрузка и проверка
Выбери плату Arduino Nano и правильный порт.

Открой Монитор порта (Ctrl+Shift+M, скорость 9600).

✅ Ты должен видеть числа, которые меняются, когда двигаешь руку.

📌 Подсказка для «смелого» задания
Внутри loop() после вычисления distance добавь:


if (distance < 20) {
  digitalWrite(13, HIGH);
} else {
  digitalWrite(13, LOW);
}
Не забудь в setup() добавить:


pinMode(13, OUTPUT);
