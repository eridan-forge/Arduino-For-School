# 🔊 Урок 8: Зуммер — голос Arduino

**Тип урока:** теория + практика  
**Время:** 40 минут  
**Уровень:** начинающий  
**Цель:** подключить пассивный зуммер, научиться генерировать звуки разной частоты, сыграть мелодию.

---

## 🧠 Теория: активный vs пассивный зуммер

| Тип | Активный | Пассивный |
|-----|----------|-----------|
| Как узнать | внутри есть плата, запирата | видна катушка, открытая плата |
| Питание | запищит сам от 5V | нужно подавать сигнал |
| Тон | один (фиксированный) | можно менять частотой |
| Команда | `digitalWrite(HIGH)` | `tone(пин, частота)` |

У нас **пассивный** — значит, мы сами управляем звуком.

---

## 🔌 Подключение (2 провода + резистор)

| Зуммер | Arduino |
|--------|---------|
| Длинная нога (+) | через резистор **100 Ом** → D3 |
| Короткая нога (-) | GND |

> ⚠️ Если зуммер пищит сам при подключении к 5V — он **активный**, и `tone()` изменит только частоту, но не выключит звук. Для практики подойдёт любой.

---

## 💻 Основные команды (шпаргалка)

```cpp
tone(пин, частота);      // включить звук заданной частоты (Гц)
tone(пин, частота, длительность); // звук на указанные миллисекунды
noTone(пин);             // выключить звук
Частоты нот (для мелодий):

Нота	Частота (Гц)

Ре (D)	294
Ми (E)	330
Фа (F)	349
Соль (G)	392
Ля (A)	440
Си (B)	494

0–3 мин | Вступление
Мы уже знаем, чем активный зуммер отличается от пассивного.
У нас пассивный — значит, мы сами можем задавать тон.
Сегодня сделаем писк, сирену и даже сыграем мелодию.

3–8 мин | Сборка схемы
Берём:

Пассивный зуммер

Резистор 100 Ом

2 провода

Подключаем строго:

длинная нога (+) → резистор → D3

короткая нога (–) → GND

8–12 мин | Первая проверка
Загружаем простейший код:

cpp
#define BUZZER 3

void setup() {
  pinMode(BUZZER, OUTPUT);
}

void loop() {
  tone(BUZZER, 1000);
  delay(500);
  noTone(BUZZER);
  delay(500);
}
✅ Слышите писк? Если нет — проверьте провода, резистор и ноги зуммера.

12–30 мин | Эксперименты со звуком
1. Меняем частоту
Вместо 1000 подставьте:

200 — низкий звук

2000 — высокий

440 — нота Ля (камертон)

2. Сирена (плавное изменение частоты)
cpp
#define BUZZER 3

void setup() {
  pinMode(BUZZER, OUTPUT);
}

void loop() {
  // нарастание
  for (int freq = 200; freq < 2000; freq += 10) {
    tone(BUZZER, freq);
    delay(5);
  }
  // затухание
  for (int freq = 2000; freq > 200; freq -= 10) {
    tone(BUZZER, freq);
    delay(5);
  }
}
3. Простая мелодия («В лесу родилась ёлочка»)
cpp
#define BUZZER 3

void setup() {
  pinMode(BUZZER, OUTPUT);
}

void loop() {
  tone(BUZZER, 262); delay(400);  // До
  tone(BUZZER, 262); delay(400);  // До
  tone(BUZZER, 294); delay(400);  // Ре
  tone(BUZZER, 330); delay(400);  // Ми
  tone(BUZZER, 330); delay(400);  // Ми
  tone(BUZZER, 294); delay(400);  // Ре
  tone(BUZZER, 262); delay(800);  // До (дольше)
  
  noTone(BUZZER);
  delay(1000);
}
4. (Для смелых) Звук, реагирующий на кнопку
cpp
#define BUZZER 3
#define BUTTON 2

void setup() {
  pinMode(BUZZER, OUTPUT);
  pinMode(BUTTON, INPUT_PULLUP);
}

void loop() {
  if (digitalRead(BUTTON) == LOW) {
    tone(BUZZER, 1000);
  } else {
    noTone(BUZZER);
  }
}
