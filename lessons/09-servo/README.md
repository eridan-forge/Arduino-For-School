## 📖 1. Введение (5 минут)

**Начать с вопроса:**

> «Ребята, как вы думаете, что общего между игрушечной машинкой на радиоуправлении, роботом-манипулятором на заводе и камерой, которая следит за вами в видеозвонке?»

*Выслушать ответы.*

**Ответ:**

> «Всё это умеет **двигаться** по команде. И сегодня мы познакомимся с устройством, которое помогает роботам двигаться точно и послушно — **сервоприводом**».

---

## 🧠 2. Что такое сервопривод (10 минут)

### Простое определение

> **Сервопривод** — это моторчик, который умеет поворачиваться на нужный угол и останавливаться. Как стрелка часов, которой можно сказать: «Покажи на 3 часа» — и она покажет.

### Аналогия для детей

> Представьте, что вы играете в игру «Замри». Я говорю: «Поверни голову налево».  
> Ваш мозг (это **контроллер**) даёт команду мышцам (это **мотор**) повернуть голову, а глаза (это **датчик положения**) смотрят, сколько вы уже повернулись.  
> Как только вы повернулись налево — вы замираете.  
> **Серва работает так же!**

### Демонстрация (показать реальную серву)

- «Вот этот синий цилиндр — это **мотор**»
- «Вот эти провода — **питание и управление**»
- «Вот этот зубчатый вал — будет **поворачиваться**»
- «Внутри есть маленький **потенциометр** (как ручка громкости), который следит за углом поворота»

---

## 🌍 3. Где мы встречаемся с сервами (10 минут)

### Пример 1: Робот-пылесос
> «Когда робот-пылесос упирается в стену, он поворачивает колёса с помощью сервоприводов, чтобы поехать в другую сторону.»

### Пример 2: Дрон (квадрокоптер)
> «У дрона нет штурвала. Он управляет наклоном с помощью серв, которые поворачивают лопасти. Тысячи раз в секунду!»

### Пример 3: Камера слежения (PTZ)
> «Камеры на стадионах могут поворачиваться и следить за мячом. Внутри — два сервопривода: один для поворота влево-вправо, второй — вверх-вниз.»

### Пример 4: Робо-рука (финальный проект)
> «А в нашей робо-руке, которую мы сделаем через 2 года, будет **5–6 сервоприводов**. Каждый отвечает за один палец или за поворот запястья. Сервы — это мышцы робота.»

---

## 🎮 4. Как мы будем управлять сервой (5 минут)

### Схема на доске
[Кнопка] → [Arduino] → [Серва] → [Поворот]
[Потенциометр] → [Arduino] → [Серва] → [Поворот]
[Ультразвуковой датчик] → [Arduino] → [Серва] → [Открыть шлагбаум]

text

> «Arduino — это мозг. Она получает сигнал от кнопки или датчика и говорит серве: *"Повернись на 90 градусов"*. Серва слушается.»

### Код — это язык, на котором мы говорим с Arduino

> «Мы напишем что-то вроде: *"Если нажата кнопка, то поверни серву на 180 градусов"*. Звучит сложно, но на самом деле это 5 строчек.»

# 🔧 Урок 9.1 : Первое знакомство с сервоприводом (практика)

## 🔌 Часть 1: Тест сервы (10 минут)

### Подключение (показать на доске/экране)

| Серва | Arduino |
|-------|---------|
| Коричневый (GND) | GND |
| Красный (VCC) | 5V |
| Жёлтый (Signal) | D9 |

> ⚠️ **Важно:** серва подключается **напрямую к пинам**, не на макетку! Провода у неё уже с «мама» разъёмами.

### Код для теста

```cpp
#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(9);
}

void loop() {
  myServo.write(0);
  delay(1000);
  myServo.write(90);
  delay(1000);
  myServo.write(180);
  delay(1000);
}
Что делаем
Подключаем серву

Загружаем код

Наблюдаем: серва крутится 0° → 90° → 180° → повтор

Разбор
#include <Servo.h> — подключаем библиотеку

Servo myServo; — создаём объект «серва»

myServo.attach(9); — говорим, что серва на пине 9

myServo.write(угол); — поворачиваем

Вопрос: «Что будет, если поменять углы на 0, 45, 90?»

🔘 Часть 2: Управление с кнопки (20 минут)
Добавляем кнопку на макетку
Схема (с внутренним резистором, без внешнего):

Кнопка: один контакт → 5V, другой → D2

Никаких резисторов не нужно, используем INPUT_PULLUP

Или с резистором (для наглядности):

Кнопка: один контакт → 5V, другой → D2

Резистор 10 кОм: D2 → GND

Код
cpp
#include <Servo.h>

Servo myServo;
int buttonPin = 2;

void setup() {
  myServo.attach(9);
  pinMode(buttonPin, INPUT_PULLUP);  // если без резистора
  // pinMode(buttonPin, INPUT);      // если с резистором
}

void loop() {
  if (digitalRead(buttonPin) == LOW) {  // LOW = нажата (INPUT_PULLUP)
    myServo.write(180);
  } else {
    myServo.write(0);
  }
}
Задания для учеников
Базовое:

Собрать схему

Загрузить код

Проверить: кнопка нажата → серва на 180°, отпустили → на 0°

Дополнительные (для тех, кто сделал быстро):

Изменить углы: пусть на 90° и обратно на 0°

Добавить третий угол: нажал → 0°, нажал → 90°, нажал → 180° (циклично)

Добавить светодиод: когда серва на 180° — LED горит

🎚️ Часть 3: Управление с потенциометра (20 минут)
Добавляем потенциометр
Потенциометр	Arduino
Левый вывод	GND
Правый вывод	5V
Средний вывод	A0
Код
cpp
#include <Servo.h>

Servo myServo;
int potPin = A0;
int angle;

void setup() {
  myServo.attach(9);
}

void loop() {
  angle = analogRead(potPin);        // читаем 0–1023
  angle = map(angle, 0, 1023, 0, 180); // превращаем в 0–180
  myServo.write(angle);
  delay(15);                         // плавность
}
Что происходит
Крутишь ручку потенциометра

Серва повторяет угол

Вопрос: «Почему delay(15)? Что будет без него?» (серва будет дёргаться, потому что команды приходят слишком быстро)

🧪 Задания для самостоятельной работы
Ученики выбирают одно или делают все:

Задание 1: Стрелка
Надеть на серву хорн (белую насадку) и примотать полоску картона. Теперь видно, куда показывает серва.

Задание 2: Указатель уровня

for (int i = 0; i <= 180; i++) {
