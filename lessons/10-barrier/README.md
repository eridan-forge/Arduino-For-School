## 🧠 2. Как работает автоматический шлагбаум (5 минут)

### Схема на доске
[Машина подъехала] → [Датчик увидел] → [Arduino принял решение] → [Серва открыла] → [Машина проехала] → [Датчик никого не видит] → [Серва закрыла]

text

### Простыми словами

> «Шлагбаум — это робот, который делает всего **три вещи**:  
> 1. **Смотрит** — есть ли кто перед ним?  
> 2. **Думает** — что делать: открыть или закрыть?  
> 3. **Действует** — крутит мотор.»

### Для детей

> «Представьте, что шлагбаум — это охранник. У него есть глаза (датчик) и рука (серва). Если охранник видит машину — он поднимает руку. Если никого нет — опускает.»

## 🔧 3. Первая проблема: почему шлагбаум дёргается (5 минут)

### Проблема

> «Если написать простой код: *"если машина близко — открой, если далеко — закрой"*, то шлагбаум будет дёргаться. Почему?»

### Объяснение

> «Представьте, что машина стоит ровно на границе. Датчик показывает: 31 см → закрыто, 29 см → открыто. Машина чуть-чуть дрогнула — и шлагбаум начинает дёргаться: открыть-закрыть-открыть-закрыть.»

### Решение №1: Флаг состояния

> «Мы вводим **флаг** — переменную, которая помнит: *«я уже открыт»* или *«я закрыт»*. Тогда команда "открыть" выполняется **только один раз**, а не каждую миллисекунду.»

**Аналогия:** «Это как табличка на двери: "ОТКРЫТО". Пока она висит, ты не пытаешься открыть дверь снова.»

## 🐢 4. Вторая проблема: резкое движение (3 минуты)

### Проблема

> «Если просто дать команду `myServo.write(90)`, шлагбаум **дёрнется** и резко откроется. Это некрасиво и может сломать механику.»

### Решение №2: Плавное движение

> «Вместо одного большого прыжка, мы делаем **90 маленьких шагов** по 1 градусу. Между шагами ставим маленькую задержку — и шлагбаум открывается плавно.»

**Для детей:** «Это как если ты не прыгаешь с лестницы, а спускаешься по ступенькам. Каждая ступенька — это 1 градус.»

```cpp
for (int angle = 0; angle <= 90; angle++) {
  myServo.write(angle);
  delay(10);
}
🎯 5. Третья проблема: чувствительность (3 минуты)
Проблема
«Рука никогда не стоит идеально ровно. Она чуть-чуть дрожит. Датчик это видит и думает, что машина то приближается, то отдаляется.»

Решение №3: Зона нечувствительности (гистерезис)
«Мы делаем два порога: открывается шлагбаум на расстоянии 35 см, а закрывается только на 50 см. Это даёт запас в 15 см, где ничего не происходит.»

Для детей: «Представьте, что дверь открывается, когда ты подошёл на 3 шага, а закрывается, когда отошёл на 5 шагов. Это чтобы ты не застрял в дверях.»

💻 Часть 3: Полный код
cpp
#include <Servo.h>

Servo myServo;

const int trig = 10;
const int echo = 11;
const int servoPin = 9;
const int redLed = 13;
const int greenLed = 12;
const int buzzer = 8;

bool isOpen = false;

// НАСТРОЙКИ (меняй здесь)
const int moveDelay = 15;      // задержка между шагами (мс)
const int startAngle = 0;      // закрыто (0°)
const int endAngle = 90;       // открыто (90°)

void setup() {
  pinMode(trig, OUTPUT);
  pinMode(echo, INPUT);
  pinMode(redLed, OUTPUT);
  pinMode(greenLed, OUTPUT);
  pinMode(buzzer, OUTPUT);
  
  myServo.attach(servoPin);
  myServo.write(startAngle);
  digitalWrite(redLed, HIGH);
  digitalWrite(greenLed, LOW);
  
  delay(500);
}

void openBarrier() {
  for (int angle = startAngle; angle <= endAngle; angle++) {
    myServo.write(angle);
    delay(moveDelay);
  }
}

void closeBarrier() {
  for (int angle = endAngle; angle >= startAngle; angle--) {
    myServo.write(angle);
    delay(moveDelay);
  }
}

int getDistance() {
  digitalWrite(trig, LOW);
  delayMicroseconds(2);
  digitalWrite(trig, HIGH);
  delayMicroseconds(10);
  digitalWrite(trig, LOW);
  
  long duration = pulseIn(echo, HIGH);
  int cm = duration * 0.034 / 2;
  return cm;
}

void loop() {
  int distance = getDistance();
  
  if (distance < 30 && distance > 0 && !isOpen) {
    openBarrier();
    digitalWrite(redLed, LOW);
    digitalWrite(greenLed, HIGH);
    tone(buzzer, 1000, 200);
    isOpen = true;
  }
  
  if ((distance >= 30 || distance == 0) && isOpen) {
    closeBarrier();
    digitalWrite(redLed, HIGH);
    digitalWrite(greenLed, LOW);
    isOpen = false;
  }
  
  delay(50);
}
