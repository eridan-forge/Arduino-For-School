# 🎛️ Урок 11: Управление сервой с потенциометра

**Тип урока:** теория + практика  
**Время:** 20–25 минут  
**Уровень:** средний (уже знают серву и потенциометр по отдельности)

---

## 🎯 Цели урока

После урока ученики:

- поймут, как читать аналоговый сигнал (`analogRead`)
- научатся превращать числа 0–1023 в углы 0–180 (`map`)
- смогут управлять сервой плавно, как джойстиком

---

## 🧠 Теория (5–7 минут)

> **Потенциометр** — это «крутилка», которая меняет напряжение. Arduino читает это напряжение и превращает в число от **0 до 1023**.  
> **Серва** понимает только углы от **0 до 180**.  
> Значит, нужно перевести 0–1023 в 0–180. Это делает функция `map()`.

**Главная идея:** крутишь ручку → серва повторяет положение.

---

## 🔌 Схема подключения

| Потенциометр | Arduino |
|--------------|---------|
| Левый вывод | GND |
| Средний вывод | A0 |
| Правый вывод | 5V |

| Серва | Arduino |
|-------|---------|
| Коричневый (GND) | GND |
| Красный (VCC) | 5V |
| Жёлтый/оранжевый (Signal) | D9 |

> ⚠️ Питание сервы — от Arduino. Для одной сервы этого достаточно.

---

## 💻 Код (полный, с комментариями)

```cpp
#include <Servo.h>

Servo myServo;      // создаём объект "серва"
int potPin = A0;    // потенциометр на пине A0

void setup() {
  myServo.attach(9);  // серва на пине D9
}

void loop() {
  int raw = analogRead(potPin);           // читаем 0–1023
  int angle = map(raw, 0, 1023, 0, 180); // превращаем в 0–180
  myServo.write(angle);                   // поворачиваем серву
  delay(15);                              // плавность
}
🧪 Задания для учеников
Базовые
Плавное управление — покрути потенциометр. Серва следит за рукой.

Ограничить угол — измени map(raw, 0, 1023, 0, 90). Что изменилось?

Реверс — поменяй 0, 180 на 180, 0. Почему серва пошла в другую сторону?

Продвинутые
Два потенциометра → одна серва — добавь второй потенциометр (на A1) и сделай так, чтобы серва крутилась только если крутить оба (логика И).

Пример для задания 4
cpp
int raw1 = analogRead(A0);
int raw2 = analogRead(A1);
int avg = (raw1 + raw2) / 2;
int angle = map(avg, 0, 1023, 0, 180);

Что менять:

Потенциометр можно подключить к любому аналоговому пину (A0–A5)

Серву — к любому ШИМ-пину (D3, D5, D6, D9, D10, D11)

В map() можно менять диапазоны углов

# 🦀 Урок 11: Клешня из двух серв

**Тип урока:** практика

## 🎯 Цели урока

После урока ученики:

- соберут клешню из двух серв и двух потенциометров
- научатся управлять двумя сервами независимо
- попробуют симметричное управление (один потенциометр на две сервы)
- поймут, как устроена механика робо-руки

---

## 🧠 Теория (8–10 минут)

> Это уже **введение в робо-руку**.

- У клешни две «губки». Каждой управляет **своя серва**.
- Каждая серва — **свой потенциометр**.
- Крутишь левую ручку → двигается левая губка.
- Крутишь правую → двигается правая губка.
- Когда обе губки сводятся вместе — клешня **закрывается**.

**Симметричный режим:** один потенциометр управляет **одновременно двумя сервами** (встречное движение).

---

## 🔌 Схема подключения

### Потенциометры
| Потенциометр 1 | Arduino |
|----------------|---------|
| Левый вывод | GND |
| Средний вывод | A0 |
| Правый вывод | 5V |

| Потенциометр 2 | Arduino |
|----------------|---------|
| Левый вывод | GND |
| Средний вывод | A1 |
| Правый вывод | 5V |

### Сервы
| Серва 1 | Arduino |
|---------|---------|
| Коричневый (GND) | GND |
| Красный (VCC) | 5V |
| Жёлтый (Signal) | D9 |

| Серва 2 | Arduino |
|---------|---------|
| Коричневый (GND) | GND |
| Красный (VCC) | 5V |
| Жёлтый (Signal) | D10 |

> ⚠️ **Питание:** Для двух серв желательно использовать **Arduino Mega** или внешний источник 5V (2А). Nano может не вытянуть.

## 💻 Код для двух независимых серв

```cpp
#include <Servo.h>

Servo servoLeft;
Servo servoRight;

const int potLeft = A0;
const int potRight = A1;

void setup() {
  servoLeft.attach(9);
  servoRight.attach(10);
}

void loop() {
  int rawL = analogRead(potLeft);
  int angleL = map(rawL, 0, 1023, 0, 180);
  servoLeft.write(angleL);

  int rawR = analogRead(potRight);
  int angleR = map(rawR, 0, 1023, 0, 180);
  servoRight.write(angleR);

  delay(15);
}
💻 (Дополнительно) Симметричная клешня — один потенциометр
cpp
#include <Servo.h>

Servo servoLeft;
Servo servoRight;
int potPin = A0;

void setup() {
  servoLeft.attach(9);
  servoRight.attach(10);
}

void loop() {
  int raw = analogRead(potPin);
  int angle = map(raw, 0, 1023, 0, 180);
  servoLeft.write(angle);
  servoRight.write(180 - angle); // встречное движение
  delay(15);
}
🛠️ Как сделать картонные «губки»
Надень на сервы хорны (белые насадки) и прикрути винтами

Вырежи из картона две полоски (10×2 см)

Сделай в них отверстие под штырь хорна

Надень полоски на хорны (или примотай скотчем)

Закрепи сервы на основании (фанера/коробка) так, чтобы полоски встречались при угле 0° и расходились при 90°
